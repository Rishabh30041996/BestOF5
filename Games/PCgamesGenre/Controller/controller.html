5: Bayonetta

Genre :AdventureAction

Publisher : Sega Nintendo

Platforms : PlayStation 3, Xbox 360, Wii U, Windows, Nintendo Switch, PlayStation 4, Xbox One

First release : October 29, 2009(Bayonetta)

Latest release: September 20, 2014(Bayonetta 2)

About Game : Bayonetta. The last survivor of an ancient witch clan who keep the balance between light, dark and chaos. Thrust straight into battle, with only one clue to her past, Bayonetta must discover the truth and fight for the future. Her daunting conquest sees her face off against countless angelic enemies and giant foes in a game of 100% pure action.

Video Card : Radeon HD6950 / GeForce GTX 570 (VRAM 768MB)

3: Call of Duty: Warzone

Genre :Battle RoyalActionMultiplayerRPG

Publisher : ActivisionLtd.

Platforms : PlayStation 4, Xbox One, Microsoft Windows

First release : March 10, 2020

About Game : The Battle Royale mode is similar to other titles in the genre where players compete in a continuously shrinking map to be the last player remaining. Players parachute onto a large game map, where they encounter other players. As the game progresses and players are eliminated, the playable area shrinks forcing the remaining players into tighter spaces. In Warzone, the non-playable areas become contaminated with a green gas that depletes health and eventually kills the player if they do not return to the safe playable area.

Minimum System Requirements:

OS : Windows 7 64-Bit or later

Processor : CPU: Intel® Core™ i3 3225 3.3 GHz or AMD Ryzen™ 5 1400

Memory : 8 GB RAM

Video Card : NVIDIA® GeForce® GTX 660 @ 2 GB / GTX 1050 or ATI® Radeon™ HD 7850 @ 2GB / AMD RX 550

Sound Card: DirectX Compatible

HDD Space : 90 GB available space

Network : Broadband Internet connection

2: PlayerUnknown's Battlegrounds

Genre :Battle RoyaleActionRPG

Publisher : PUBG Corporation

Platforms : Microsoft Windows, Android, iOS, Xbox One, PlayStation 4, Stadia

First release : December 20, 2017

About Game : PLAYERUNKNOWN'S BATTLEGROUNDS is a battle royale shooter that pits 100 players against each other in a struggle for survival. Gather supplies and outwit your opponents to become the last person standing. Every few minutes, the playable area of the map begins to shrink down towards a random location, with any player caught outside the safe area taking damage incrementally, and eventually being eliminated if the safe zone is not entered in time.

Minimum System Requirements:

OS :64-bit Windows 7, Windows 8.1, Windows 10

Processor : Intel Core i5-4430 / AMD FX-6300

Memory : 8 GB RAM

Video Card : NVIDIA GeForce GTX 960 2GB / AMD Radeon R7 370 2GB

Directx: Version 11

HDD Space : 30 GB available space

Network : Broadband Internet connection

1: Fortnite Battle Royale

Genre :Battle RoyaleMultiplayerRPG

Publisher : Epic Games

Platforms : Windows, macOS, Nintendo Switch, PlayStation 4, PlayStation 5, Xbox One, Xbox Series X/S, iOS, Android

First release : September 26, 2017

About Game : The game normally is played either with each player on their own, or in a squad of two to four players, with up to 100 players participating each round. The round starts with players, weaponless, skydiving from floating buses ("Battle Bus") then deploying a glider onto a region of land. The primary goal is to be the last player or team alive by eliminating or avoiding other players. When playing in solo modes, players are immediately eliminated when they exhaust their health.

Minimum System Requirements:

OS :Windows 7/8/10 64-bit

Processor : Core i3 2.4 Ghz

Memory : 8 GB RAM

Video Card : Nvidia GTX 660 or AMD Radeon HD 7870 equivalent DX11 GPU

DirectX: Version 11

HDD Space : 32 GB

Network : Broadband Internet connection
