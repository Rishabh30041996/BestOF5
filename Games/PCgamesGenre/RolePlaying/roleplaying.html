Best Five PC Role-Playing Games

5: Wasteland

Genre :Role-PlayingMultiplayer

Publisher : Electronic Arts inXile Entertainment, Xbox Game Studios

Platforms : Apple II, Commodore 64, MS-DOS, Microsoft Windows, OS X, Linux, Xbox One

First release : January 2, 1988(Wasteland)

Latest release: 28 Aug, 2020(Wasteland 3)

About Game : In Wasteland 3 you take command of a squad of Desert Rangers, lawmen and women in a post-nuclear world, trying to rebuild society from the ashes. More than a century after the bombs fell, you’re fighting a losing battle to keep your beloved Arizona alive. Then the self-proclaimed Patriarch of Colorado radios, promising aid if you'll do a job he can only entrust to an outsider—rescue his land from the ambitions of his three bloodthirsty children.

Minimum System Requirements:

OS :Windows 10

Processor : Intel Core i5-3.3 GHz or better, or AMD Equivalent

Memory : 8 GB RAM

Video Card : Nvidia GTX 760 or AMD Equivalent

Sound Card: DirectX compatible sound card

HDD Space : 45 GB available space

4: The Witcher

Genre :Open WorldRPGAdventure

Publisher : CD PROJEKT RED

Platforms : Microsoft Windows, OS X

First release : 26 October 2007(The Witcher)

Latest release : 18 May, 2015(The Witcher® 3: Wild Hunt)

About Game : The Witcher: Wild Hunt is a story-driven open world RPG set in a visually stunning fantasy universe full of meaningful choices and impactful consequences. In The Witcher, you play as professional monster hunter Geralt of Rivia tasked with finding a child of prophecy in a vast open world rich with merchant cities, pirate islands, dangerous mountain passes, and forgotten caverns to explore.

Minimum System Requirements:

OS :64-bit Windows 7, 64-bit Windows 8 (8.1) or 64-bit Windows 10

Processor : Intel CPU Core i5-2500K 3.3GHz / AMD CPU Phenom II X4 940

Memory : 6 GB RAM

Video Card : Nvidia GPU GeForce GTX 660 / AMD GPU Radeon HD 7870

Sound Card: 100% DirectX 9.0 compatible

HDD Space : 35 GB available space

3: Mass Effect

Genre :Role-PlayingScience fictionMilitary science fiction

Publisher : Electronic Arts

Platforms : Xbox 360, Microsoft Windows, iOS, PlayStation 3, Android, Windows Phone, Wii U, PlayStation 4, Xbox One

First release : November 20, 2007(Mass Effect)

Latest release : March 21, 2017(Mass Effect: Andromeda)

About Game : Mass Effect™: Andromeda takes players to the Andromeda galaxy, far beyond the Milky Way. There, you'll lead the fight for a new home in hostile territory as the Pathfinder - a leader of military-trained explorers. This is the story of humanity’s next chapter, and your choices throughout the game will ultimately determine humanity's survival.

Minimum System Requirements:

OS : 64-bit Windows 7, Windows 8.1 and Windows 10

Processor : Intel Core i5 3570 or AMD FX-6350

Memory : 8 GB RAM

Video Card : NVIDIA GTX 660 2GB, AMD Radeon 7850 2GB

DirectX: Version 11

HDD Space : 55 GB available space

2: Warcraft

Genre :Real-time strategyRole-Playing

Publisher : Blizzard Entertainment

Platforms : MS-DOS, Microsoft Windows, Mac OS, Sega Saturn, PlayStation, iOS, Android

First release : November 23, 1994(Warcraft: Orcs & Humans)

Latest release : November 23, 2020(World of Warcraft: Shadowlands)

About Game : Shadowlands involves a level reduction ("level squish") with player characters at level 120 (the level cap in Battle for Azeroth) reduced to level 50, with level 60 being the new level cap (as it had been in the original game). In what Blizzard has called a "New Game+ experience", newly created characters have an updated starting experience on an island.

Minimum System Requirements:

OS :Windows® 7 64-bit

Processor : Intel® Core™ i5-3450 AMD FX™ 8300

Memory : 4GB RAM

Video Card : NVIDIA® GeForce® GTX 760 2 GB or AMD Radeon™ RX 560 2 GB or Intel® UHD Graphics 630 (45W TDP)

Directx: 9.0c

HDD Space : 100GB available space

Network : Broadband Internet connection

1: Star Wars Jedi: Fallen Order

Genre :AdventureActionRPG

Publisher : Electronic Arts

Platforms : Microsoft Windows, PlayStation 4, Xbox One, Stadia

First release : November 15, 2019

About Game : Star Wars Jedi: Fallen Order takes place five years after the events of Revenge of the Sith. The game follows Jedi Padawan Cal Kestis (Cameron Monaghan) as he is being hunted by the Second Sister (Elizabeth Grullon) and her partner, the Ninth Sister (Misty Lee), both trained by the Sith Lord Darth Vader (Scott Lawrence) as part of the Imperial Inquisitors program.

Minimum System Requirements:

OS :64-bit Windows 7/8.1/10

Processor : AMD FX-6100/Intel i3-3220 or Equivalent

Memory : 8 GB RAM

Video Card : AMD Radeon HD 7750, NVIDIA GeForce GTX 650 or Equivalent

DirectX: Version 11

HDD Space : 55 GB available space
